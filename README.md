# Creatorly Video Lab Backend

This is the backend service for analyzing Instagram Reel drafts. It uses `ffmpeg` for video processing and `Gemini 1.5 Flash` for AI visual analysis.

## Deploying to Railway

1. Sign up at [Railway.app](https://railway.app/).
2. You can use the Railway CLI to deploy this folder directly:
   ```bash
   npm i -g @railway/cli
   railway login
   railway init
   railway up
   ```
3. Once deployed, go to the Railway dashboard, click on your project, go to **Variables**, and add your Gemini API Key:
   - `GEMINI_API_KEY=your_new_api_key_here`
4. Copy the public domain URL generated by Railway (e.g., `https://creatorly-videolab-production.up.railway.app`)
5. Update `public/js/video-lab.js` in your `creatorly-ai` Vercel project to point `API_URL` to this new Railway URL.

## Local Testing
```bash
npm install
npm run dev
```
